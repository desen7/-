Общая информация Стек: Java, Swing (Graphics2D), ООП, Паттерны проектирования

Описание: Учебный проект, представляющий собой графическое приложение с анимацией. Программа визуализирует динамичную сцену природы (небо, солнце, облака, дом, дерево) в реальном времени.

Ключевая функциональность и особенности:

Анимация объектов: Солнце движется по эллиптической траектории (имитация восхода/заката), три облака перемешаются горизонтально с разной скоростью.

Графика: Использование градиентов, сглаживания (антиалиасинг), процедурной генерации (трава) и перекрывающихся фигур (облака, крона) для создания визуальной глубины.

Архитектура: Проект построен на принципах SOLID и ООП (инкапсуляция, наследование, полиморфизм). Ключевым элементом является интерфейс Drawable, который обеспечивает единообразную обработку всех объектов сцены.

Управление анимацией: Реализован паттерн Наблюдатель (Observer) через Swing Timer в специальном контроллере (AnimationController), который обновляет состояние объектов и перерисовывает сцену каждые 50 мс (20 FPS).

Дизайн: Централизованное управление цветами (ColorPalette) и четкое разделение логики отрисовки (DrawingPanel) от моделей объектов.

Цель проекта: Демонстрация практического применения ООП, работы с графикой и анимацией в Java, а также создания модульной и расширяемой архитектуры.# -
